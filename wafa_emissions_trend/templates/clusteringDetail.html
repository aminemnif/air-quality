Clustering Analysis of Emission Patterns

Overview

This project involves identifying emission patterns across states and sectors in the USA using clustering algorithms. The analysis provides actionable insights for policymakers to address key environmental issues.

Data Used

The analysis was conducted using publicly available EPA datasets containing detailed emission data for various pollutants, states, and sectors.

  • Timeframe: 2002 to 2022
  • Pollutants: CO2, VOCs, PM2.5, PM10, and more
  • Sectors: On-road sources, industrial emissions, wildfires, and others

Methods

The project employed clustering techniques to identify patterns:

  • K-Means Clustering: Used to group states and sectors based on emission similarity.
  • Agglomerative Clustering: Provided hierarchical insights into relationships between sectors and states.

Findings

The analysis revealed the following key insights:

  • Clusters highlighted regional similarities in industrial emissions and their contributors.

Impact

The results of this analysis enable policymakers to prioritize sectors and regions for targeted mitigation efforts, promoting cleaner air and a sustainable environment.
